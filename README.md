# Pythagorean_Triple
Given a number n, find a Pythagorean Triple for a given sum.

A Pythagorean Triple is a set of intergers a, b and c such that a^2 + b^2 = c^2.

Examples:

1) Input: 12
   Output: 3, 4, 5
   
2) Input: 56
   Output: 7, 24, 25
   
3) Input: 7
   Output: No Triple
